
# map

  apply a transformation to each item within an object

## Getting Started

_With component_  

	$ component install jkroso/map

_With npm_  

	$ npm install --save https://github.com/jkroso/map/archive/master.tar.gz


then in your app:

```js
var map = require('map')
```
